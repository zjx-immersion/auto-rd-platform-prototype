# 自动研发平台原型

端到端智能研发协同平台原型系统

## 🚀 一键启动

```bash
./start.sh
```

服务将在端口 **6060** 启动

## 📍 访问地址

- **本地访问**: http://localhost:6060
- **网络访问**: http://<你的IP>:6060

## 🔐 默认登录信息

```
用户名: admin
密码: admin123
```

## 📦 手动启动

如果需要手动启动：

```bash
cd frontend
npm install  # 仅首次需要
npm run dev
```

## 🎯 功能验证指南

### 三种导航模式

1. **流程驱动导航** - 基于研发流程的引导式体验
   - 我的流程
   - P1-P6 六大流程阶段

2. **固有功能导航** - 按功能域分类的传统导航
   - C0: 领域项目管理
   - C1: 需求管理
   - C2: 资产管理
   - C3: 规划协调
   - C4: 迭代执行
   - C5: 质量保证
   - C6: 发布管理
   - C7: 度量分析

3. **工作台导航** - 个性化的工作中心
   - 我的工作台
   - 团队工作台
   - 项目工作台
   - 管理工作台

### 推荐验证路径

#### 路径A: 流程驱动体验
```
登录 → 流程驱动 → 我的流程 → P1战略规划流程 → 
创建Epic → 完成流程向导
```

#### 路径B: 固有功能体验
```
登录 → 固有功能 → C1需求管理 → Epic管理 → 
Epic列表 → 查看详情 → 创建新Epic
```

#### 路径C: 工作台体验
```
登录 → 工作台 → 我的工作台 → 
查看待办事项/日程安排/统计图表
```

## 📁 项目结构

```
.
├── start.sh                    # 一键启动脚本
├── frontend/                   # Vue 3 前端项目
│   ├── src/
│   │   ├── components/        # 可复用组件
│   │   │   ├── Layout/       # 布局组件（Shell、导航等）
│   │   │   └── Common/       # 通用组件
│   │   ├── views/            # 页面视图
│   │   │   ├── C0-C7/       # 功能域页面
│   │   │   ├── Process/     # 流程页面
│   │   │   └── Workspace/   # 工作台页面
│   │   ├── router/          # 路由配置
│   │   ├── stores/          # 状态管理 (Pinia)
│   │   └── types/           # TypeScript 类型定义
│   └── package.json
├── platform-design/           # 平台设计文档
├── page-design/              # 页面设计文档
├── prototype-framework/      # 原型框架设计
└── analysis-workspace/       # 分析报告

```

## 🛠️ 技术栈

- **框架**: Vue 3 + TypeScript
- **构建工具**: Vite 5
- **UI组件**: Element Plus
- **状态管理**: Pinia
- **路由**: Vue Router 4
- **样式**: SCSS
- **图表**: ECharts

1. **三模式导航** - 满足不同用户习惯和场景

4. **模块化架构** - 易于扩展和维护
