# C2-F03: 复用评估

> **功能编号**: C2-F03  
> **功能名称**: 复用评估  
> **所属模块**: C2-资产管理 > 资产搜索与匹配  
> **主要角色**: SO (Software Owner), PM (Project Manager)  
> **价值流阶段**: S3-方案设计

---

## 1. 功能概述

### 1.1 功能描述
对候选资产进行全面的复用可行性评估，包括技术可行性、成本效益分析、风险评估等，生成详细的评估报告，辅助复用决策。

### 1.2 业务价值
- 量化复用收益，支持数据驱动决策
- 识别复用风险，降低项目风险
- 评估适配成本，优化资源分配

### 1.3 使用场景
- SO评估资产复用的可行性和收益
- PM评估资产复用对项目进度和成本的影响
- 团队评审资产复用方案

## 3. 数据字段

### 3.1 评估输入

| 字段名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| assetId | String | 是 | 资产ID |
| requirementId | String | 是 | 需求ID |
| targetEnvironment | Object | 否 | 目标环境信息 |
| projectContext | Object | 否 | 项目上下文信息 |

### 3.2 评估结果

| 字段名 | 类型 | 说明 |
|--------|------|------|
| evaluationId | String | 评估ID |
| overallScore | Float | 综合评分(0-100) |
| recommendation | Enum | 复用建议: HIGHLY_RECOMMENDED/RECOMMENDED/CONDITIONAL/NOT_RECOMMENDED |
| estimatedSaving | Float | 预计节省比例(0-100) |
| adaptationEffort | Integer | 适配工作量(人天) |
| riskLevel | Enum | 风险等级: LOW/MEDIUM/HIGH |
| technicalFeasibility | Object | 技术可行性评估 |
| costBenefitAnalysis | Object | 成本效益分析 |
| riskAssessment | Object | 风险评估 |
| dependencies | Array | 依赖分析 |

## 7. 交互设计

### 7.1 评估流程

1. **选择资产和需求**
   - 从资产匹配页面进入
   - 或手动选择资产和需求

2. **执行自动评估**
   - 系统自动分析技术可行性
   - 计算成本效益
   - 识别风险

3. **查看评估报告**
   - 切换不同Tab查看详情
   - 查看评估详情、成本效益、风险分析

4. **做出复用决策**
   - 基于评估报告做出决策
   - 选择"确认复用"或"放弃复用"

### 7.2 报告导出

1. 点击"导出评估报告"
2. 选择导出格式(PDF/Word/Excel)
3. 生成并下载报告
