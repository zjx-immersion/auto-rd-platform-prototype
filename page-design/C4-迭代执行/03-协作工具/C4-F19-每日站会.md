# C4-F19: 每日站会

> **功能编号**: C4-F19  
> **功能名称**: 每日站会  
> **所属模块**: C4-迭代执行 > 协作工具  
> **主要用户**: DL/DEV  
> **页面类型**: 会议页 + 表单页

---

## 一、功能概述

### 1.1 功能定位

每日站会是Scrum的核心实践，团队每天同步进度、识别阻塞、协调工作。

### 1.2 核心价值

- **快速同步**: 15分钟快速同步团队进度
- **识别阻塞**: 及时发现和解决阻塞问题
- **团队协作**: 增强团队沟通和协作
- **透明可视**: 提高工作透明度

### 1.3 三个问题

1. **昨天做了什么?** - 回顾昨天的工作
2. **今天计划做什么?** - 计划今天的工作
3. **遇到什么阻塞?** - 识别需要帮助的问题

## 三、数据设计

```typescript
interface DailyStandup {
  id: string;
  sprintId: string;
  date: Date;
  meetingInfo: {
    startTime: Date;
    endTime: Date;
    location: string;
    host: string;
    attendees: string[];
  };
  reports: Array<{
    userId: string;
    userName: string;
    yesterday: string[];
    today: string[];
    blockers: Array<{
      description: string;
      help: string;
    }>;
    submitted: boolean;
    submittedAt?: Date;
  }>;
  actionItems: Array<{
    description: string;
    assignee: string;
    dueDate: Date;
  }>;
  minutes: string;
  status: 'SCHEDULED' | 'IN_PROGRESS' | 'COMPLETED';
}
```
