# C1-F15 SSTS创建

> **功能编号**: C1-F15  
> **功能名称**: SSTS创建  
> **所属能力域**: C1-需求管理  
> **主要用户**: SE（系统工程师）、SO（软件负责人）  
> **页面类型**: 创建页

---

## 一、功能概述

### 1.1 功能定位
SSTS创建是SE/SO独立创建系统/技术/安全需求的功能，区别于从Feature拆解SSTS（C1-F14），支持直接创建独立的SSTS。

### 1.2 核心价值
- **灵活创建**: 支持独立创建SSTS，不依赖Feature
- **类型分类**: 支持系统需求、技术需求、安全需求三种类型
- **安全管理**: 支持ASIL等级标注和安全需求管理
- **标准符合**: 符合ISO 26262等汽车安全标准

### 1.3 使用场景
- 补充技术需求
- 安全需求独立创建
- 平台级需求创建
- 跨Feature共享需求

## 三、数据字段

### 3.1 基本信息字段

| 字段名 | 字段类型 | 是否必填 | 说明 |
|--------|---------|---------|------|
| title | String | 是 | SSTS标题，最多100字符 |
| featureId | String | 否 | 关联的Feature ID |
| type | Enum | 是 | SSTS类型: SYSTEM/TECHNICAL/SAFETY |
| category | Enum | 是 | 需求分类: FUNCTIONAL/PERFORMANCE/SAFETY/INTERFACE |
| priority | Enum | 是 | 优先级: P0/P1/P2 |
| owner | User | 是 | 负责人（SE/SO） |
| asilLevel | Enum | 否 | 安全等级: QM/ASIL-A/B/C/D（仅安全需求） |

### 3.2 需求内容字段

| 字段名 | 字段类型 | 是否必填 | 说明 |
|--------|---------|---------|------|
| content | RichText | 是 | 需求描述，支持富文本 |
| acceptanceCriteria | Array<String> | 是 | 验收标准列表 |
| relatedStandards | Array<String> | 否 | 相关标准（仅安全需求） |
| tags | Array<String> | 否 | 标签列表 |

### 3.3 验证方法字段

| 字段名 | 字段类型 | 是否必填 | 说明 |
|--------|---------|---------|------|
| verificationMethods | Array<Enum> | 是 | 验证方法列表 |
| testScenarios | Array<Scenario> | 否 | 测试场景列表 |
| qualityRequirements | Object | 是 | 质量要求 |
| upstreamRequirements | Array<String> | 否 | 上游需求ID列表 |

## 四、交互设计

### 4.1 类型切换

根据选择的SSTS类型，动态显示不同的字段：

```
选择类型: 安全需求 (SAFETY)
↓
显示额外字段:
- 安全等级 (ASIL Level)
- 相关标准 (Related Standards)
- 故障模式分析 (FMEA)
- 安全机制 (Safety Mechanisms)
```

基于需求描述，提供智能提示：

```

检测到您的需求涉及"车道线检测"，建议：
1. 添加验证方法: 仿真测试、实车测试
2. 参考标准: ISO 26262
3. 相关资产: ASSET-ADAS-001 车道线检测模块
```

## 五、API接口

### 5.1 创建SSTS

**请求**:
```http
POST /api/v1/ssts
Content-Type: application/json

{
  "title": "车道线检测系统需求",
  "featureId": "ADAS-F001",
  "type": "SYSTEM",
  "category": "FUNCTIONAL",
  "priority": "P0",
  "owner": {
    "id": "U004",
    "name": "赵敏"
  },
  "content": "系统应能够检测车道线，检测准确率≥95%...",
  "acceptanceCriteria": [
    "检测准确率≥95%",
    "检测距离≥50m"
  ],
  "verificationMethods": ["SIMULATION", "VEHICLE_TEST"],
  "tags": ["车道线检测", "视觉感知"]
}
```

**响应**:
```json
{
  "code": 200,
  "message": "SSTS创建成功",
  "data": {
    "id": "ADAS-S001",
    "title": "车道线检测系统需求",
    "status": "DRAFT",
    "createdAt": "2025-01-15T10:00:00Z"
  }
}
```
