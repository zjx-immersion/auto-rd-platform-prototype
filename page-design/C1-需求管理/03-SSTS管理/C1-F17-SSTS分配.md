# C1-F17: SSTS分配

> **功能编号**: C1-F17  
> **功能名称**: SSTS分配  
> **所属模块**: C1-需求管理 > SSTS管理  
> **主要角色**: SE, SO  
> **价值流阶段**: S3-方案设计

---

## 1. 功能概述

### 1.1 功能描述
将SSTS分配给SO或开发团队，支持批量分配、工作量评估、负载均衡等功能。

### 1.2 业务价值
- 合理分配开发任务
- 平衡团队工作负载
- 提高开发效率

## 3. API接口

**接口**: `POST /api/ssts/{sstsId}/assign`

**请求示例**:
```json
{
  "owner": "user-002",
  "collaborators": ["user-003", "user-004"],
  "estimatedEffort": 5,
  "plannedStartDate": "2025-01-20",
  "plannedEndDate": "2025-01-27",
  "assignmentNote": "李四负责图像预处理模块的开发，王五和赵六协助。"
}
```

## 4. 页面跳转

### 4.1 入口
- SSTS详情 > 点击"分配"
- SSTS列表 > 批量操作 > 分配

### 4.2 出口
- 保存成功 → SSTS详情页 (C1-F12)
