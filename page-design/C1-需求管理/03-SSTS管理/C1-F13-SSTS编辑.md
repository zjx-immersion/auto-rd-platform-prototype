# C1-F13: SSTS编辑

> **功能编号**: C1-F13  
> **功能名称**: SSTS编辑  
> **所属模块**: C1-需求管理 > SSTS管理  
> **主要角色**: SE, SO  
> **价值流阶段**: S2-需求分析

---

## 1. 功能概述

### 1.1 功能描述
编辑SSTS的基本信息、验收标准、安全等级等，支持变更记录和审批流程。

### 1.2 业务价值
- 灵活调整SSTS需求
- 记录变更历史
- 确保变更可追溯

## 3. API接口

**接口**: `PUT /api/ssts/{sstsId}`

**请求示例**:
```json
{
  "sstsName": "车道线图像预处理",
  "type": "功能",
  "safetyLevel": "QM",
  "owner": "user-002",
  "description": "对输入的图像进行预处理...",
  "acceptanceCriteria": [
    "图像预处理延迟 < 10ms",
    "支持多种图像格式"
  ],
  "changeReason": "调整性能指标，优化验收标准"
}
```
