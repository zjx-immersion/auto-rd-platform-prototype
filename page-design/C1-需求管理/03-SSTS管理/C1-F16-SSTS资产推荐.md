# C1-F16: SSTS资产推荐

> **功能编号**: C1-F16  
> **功能名称**: SSTS资产推荐  
> **所属模块**: C1-需求管理 > SSTS管理  
> **主要角色**: SO (Software Owner)  
> **价值流阶段**: S3-方案设计

---

## 1. 功能概述

### 1.1 功能描述
基于SSTS需求，AI智能推荐可复用的Module资产，提供匹配度分析和复用建议。

### 1.2 业务价值
- 提高Module资产复用率
- 降低开发成本
- 加速SSTS实现

## 3. API接口

**接口**: `POST /api/ssts/{sstsId}/asset-recommendations`

**响应示例**:
```json
{
  "code": 200,
  "data": {
    "recommendations": [
      {
        "assetId": "ASSET-ADAS-002",
        "assetName": "图像预处理模块 V1.0",
        "assetType": "Module",
        "matchScore": 92,
        "maturityLevel": 4,
        "matchAnalysis": {
          "functionalMatch": 95,
          "technicalMatch": 100,
          "performanceMatch": 90
        },
        "reuseRecommendation": "建议直接复用，需要少量扩展。预计节省开发时间70%。"
      }
    ]
  }
}
```

## 4. 页面跳转

### 4.1 入口
- SSTS详情 > 点击"资产推荐"

### 4.2 出口
- 点击"查看详情" → 资产详情页 (C2-F01)
- 点击"选择复用" → 关联资产并返回SSTS详情
