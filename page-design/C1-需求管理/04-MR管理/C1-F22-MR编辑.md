# C1-F22: MR编辑

> **功能编号**: C1-F22  
> **功能名称**: MR编辑  
> **所属模块**: C1-需求管理 > MR管理  
> **主要角色**: SO  
> **价值流阶段**: S3-方案设计

---

## 1. 功能概述

### 1.1 功能描述
编辑MR的基本信息、验收标准、复杂度评估等，支持变更记录和审批流程。

### 1.2 业务价值
- 灵活调整MR需求
- 记录变更历史
- 确保变更可追溯

## 3. API接口

**接口**: `PUT /api/mrs/{mrId}`

**请求示例**:
```json
{
  "mrName": "图像去噪",
  "complexity": "中",
  "estimatedEffort": 3,
  "owner": "user-001",
  "description": "实现图像去噪功能...",
  "acceptanceCriteria": [
    "去噪效果PSNR > 30dB",
    "处理延迟 < 5ms"
  ],
  "changeReason": "调整性能指标，优化验收标准"
}
```

### 4.2 出口
- 保存成功 → MR详情页 (C1-F20)
