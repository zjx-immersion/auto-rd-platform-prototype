## 二、表单字段设计

### 2.1 基础信息

| 字段名称 | 字段类型 | 是否必填 | 默认值 | 验证规则 | 说明 |
|---------|---------|---------|--------|---------|------|
| Epic ID | 文本框（只读） | 是 | 自动生成 | - | 格式：{产品代码}-E{序号} |
| Epic标题 | 文本框 | 是 | - | 长度≤100字符 | 简洁描述Epic |

| 所属产品 | 下拉选择 | 是 | - | - | 智能驾驶/智能座舱/电子电器 |
| Epic负责人 | 用户选择器 | 是 | 当前用户 | - | 默认为创建人 |

| 字段名称 | 字段类型 | 是否必填 | 默认值 | 可选值 | 说明 |
|---------|---------|---------|--------|--------|------|
| 业务价值 | 单选按钮 | 是 | MEDIUM | CRITICAL/HIGH/MEDIUM/LOW | 业务价值评估 |
| 优先级 | 单选按钮 | 是 | P2 | P0/P1/P2/P3 | 优先级 |
| MoSCoW分类 | 单选按钮 | 是 | SHOULD | MUST/SHOULD/COULD/WONT | MoSCoW分类 |
| 业务目标 | 文本域 | 否 | - | 长度≤500字符 | 业务目标描述 |

| 字段名称 | 字段类型 | 是否必填 | 默认值 | 验证规则 | 说明 |

| 目标版本 | 下拉选择 | 是 | - | - | 计划发布版本 |

| 计划结束日期 | 日期选择器 | 否 | - | ≥开始日期 | 计划结束日期 |

| 字段名称 | 字段类型 | 是否必填 | 默认值 | 验证规则 | 说明 |

## 三、页面交互

### 3.1 表单验证

**实时验证**:
- Epic标题：输入时检查长度，超过100字符提示
- Epic描述：输入时检查长度，超过2000字符提示
- 日期：选择时检查逻辑，结束日期必须≥开始日期

**提交验证**:
- 所有必填字段必须填写
- 验收标准至少填写1条
- 如果验证失败，高亮错误字段并显示错误提示

### 3.2 按钮交互

**[取消]按钮**:
- 点击后弹窗确认："确定要取消吗？未保存的内容将丢失。"
- 确认后返回Epic列表页

**[保存草稿]按钮**:
- 保存当前填写内容为草稿状态
- 提示："草稿已保存"
- 保持在当前页面

**[提交]按钮**:
- 验证所有必填字段
- 提交成功后提示："Epic创建成功"
- 跳转到Epic详情页

### 3.3 智能提示

**Epic ID自动生成**:
- 选择"所属产品"后，自动生成Epic ID
- 格式：ADAS-E001、CABIN-E001、EE-E001

**验收标准模板**:
- 提供常用验收标准模板
- 点击模板可快速插入
