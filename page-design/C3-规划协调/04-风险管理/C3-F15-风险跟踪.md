## 一、功能概述

风险跟踪负责持续监控风险状态、缓解措施执行、风险实际发生情况，确保风险得到有效控制。

### 核心价值
- ✅ 风险状态实时跟踪  ✅ 缓解措施进度监控  
- ✅ 风险触发和实现处理  ✅ 风险趋势分析  
- ✅ 风险报告自动生成

---

## 二、数据模型和API

```typescript
interface RiskTracking {
  riskId: string;
  currentStatus: string;
  currentRiskScore: number;
  trend: 'increasing' | 'stable' | 'decreasing';
  mitigationProgress: {actionId: string, progress: number}[];
  reviews: Review[];
  materialization?: Materialization;
  alerts: Alert[];
}

// API
GET /api/pi/{piId}/risks/tracking-dashboard
PUT /api/risks/{riskId}/status
POST /api/risks/{riskId}/materialize
POST /api/risks/{riskId}/close
```
